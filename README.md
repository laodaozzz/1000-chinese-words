# 1000-chinese-words

[English](#english) | [中文](#中文)

---

## English

An interactive web app for learning the 1000 most common Chinese characters. Click any character to get real-time AI-generated vocabulary, English definitions, and bilingual example sentences powered by DeepSeek API.

### Features

- Displays 1000 frequently used Chinese characters in a grid
- Click any character to generate 5 common words containing it
- Each word includes:
  - English definition
  - Chinese example sentence
  - English translation of the example
- Classical Chinese paper-style UI design
- API key stored locally in browser (localStorage)
- Single HTML file, no build step required

### Setup

1. Get a DeepSeek API key from [platform.deepseek.com](https://platform.deepseek.com)
2. Open `index.html` in a browser
3. Click the gear icon (⚙️) in the top-right corner
4. Enter your API key and save

### Usage

Simply click on any Chinese character in the grid. A popup will appear showing vocabulary related to that character, generated in real-time by DeepSeek AI.

## 中文

一个交互式 Web 应用，用于学习 1000 个最常用汉字。点击任意汉字，即可通过 DeepSeek API 实时生成包含该字的词语、英文释义和双语例句。

### 功能特点

- 以网格形式展示 1000 个高频汉字
- 点击汉字生成 5 个包含该字的常用词语
- 每个词语包含：
  - 英文释义
  - 中文例句
  - 例句英文翻译
- 古典纸张风格 UI 设计
- API Key 存储在浏览器本地（localStorage）
- 单个 HTML 文件，无需构建

### 使用准备

1. 从 [platform.deepseek.com](https://platform.deepseek.com) 获取 DeepSeek API Key
2. 在浏览器中打开 `index.html`
3. 点击右上角齿轮图标 (⚙️)
4. 输入 API Key 并保存

### 使用方法

直接点击网格中的任意汉字，弹窗将实时显示由 DeepSeek AI 生成的相关词汇。
